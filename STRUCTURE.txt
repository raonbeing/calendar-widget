# 파일 구조

## 소스 파일

| 파일 | 역할 |
|------|------|
| `calendar_widget.py` | 진입점 (main) — 실행 시 이 파일만 호출 |
| `src/ui.py` | tkinter UI 전체 (CalendarWidget 클래스, 이벤트 렌더링, 다이얼로그) |
| `src/api.py` | Google Calendar API 호출 (인증, 일정 조회, 일정 생성, 일정 삭제) |
| `src/config.py` | 사용자 설정 저장·로드 (투명도, 배경색, 강조색) |

## secrets/ 폴더 (git 제외 — 절대 커밋 금지)

| 파일 | 설명 |
|------|------|
| `secrets/credentials.json` | Google OAuth 클라이언트 ID — Cloud Console에서 직접 발급 후 수동 배치 |
| `secrets/token.json` | OAuth 인증 토큰 — 최초 실행 시 자동 생성 |
| `secrets/config.json` | 투명도 등 사용자 설정 — 슬라이더 조작 시 자동 저장 |
