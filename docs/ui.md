# UI 기능 (2026-06-05 기준)

tkinter Canvas 기반 24시간 타임라인 뷰.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 24h 타임라인 | 일정 블록을 시간 비례로 캔버스에 표시 |
| 00-06시 구간 접힘 | 기본 숨김, 클릭 시 펼침 (새벽 시간대 공간 절약) |
| 일정 삭제 | 블록 클릭 → 확인 다이얼로그 → Google Calendar에서 삭제 |
| 설정 다이얼로그 | 투명도, 배경색(`bg_widget`), 강조색(`acc_event`, `acc_allday`) 조절 |
| 설정 자동 저장 | `secrets/config.json`에 저장 |

## 구조

- `CalendarWidget` 클래스가 UI 전체를 담당 (`src/ui.py`)
- 색상 설정은 `src/config.py`에서 로드
- 리스트 뷰 → 타임라인 뷰로 전환한 이유: 시간 흐름 파악이 어려웠음
